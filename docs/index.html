This launcher is an alternative for PSO2 Tweaker (Website).

This pre-release is pretty much like BETA, currently the launcher can only:

  • Install and Update PSO2 (Game's requisition excluded) (Support cache)
  • Check for English Patches updates. (Raiser system included) (Support cache)
  • Install English Patches. (Raiser system included) (Support cache)
  • Uninstall English Patches. (Raiser system included) (Support cache)
  • Manage PSO2 Plugins.
  • Manage PSO2 Settings.
  • Manage PSO2Proxy and Telepipe. (Both install and uninstall PSO2Proxy/Telepipe)
  • Launch PSO2 Client.
